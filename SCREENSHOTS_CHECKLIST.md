## How to Capture

**Desktop:**
1. Open Chrome/Safari
2. Navigate to URL
3. Cmd+Shift+4 (macOS) or Snipping Tool (Windows)
4. Full page screenshots recommended

**Browser DevTools:**
- For mobile: Chrome DevTools → Device toolbar
- For full page: Chrome → Cmd+Shift+P → "Capture full size screenshot"
